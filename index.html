Grid

Small, simple, powerful

Grid is a simple, easy-to-use responsive CSS grid system that enables you to tailor your content to multiple devices.

Screen Sizes

You are viewing this page on a small medium large screen.

Grid supports three different screen sizes, identified by a single-character prefix: small, medium or large. You can think of these as corresponding to mobile, tablet and desktop respectively.

To show or hide an element when at a specific size, use the X-show and X-hide classes, where X is the relevant size prefix. Elements marked as X-show will be displayed only at the specified size; elements with X-hide will be displayed at every size except the one specified.
