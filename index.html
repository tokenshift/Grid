Grid

Small, simple, powerful

A simple, responsive CSS grid that enables you to easily tailor your content to multiple devices.

Screen Sizes

You are viewing this page on a small medium large screen. Go ahead and resize the window to watch the layout adjust.

Grid supports three different screen sizes, identified by a single-character prefix: small, medium or large. You can think of these as corresponding to mobile, tablet and desktop respectively.

To show or hide an element when at a specific size, use the .X-show and .X-hide classes, where X is the relevant size prefix. Elements marked as .X-show will be displayed only at the specified size; elements with .X-hide will be displayed at every size except the one specified.

Usage

Unlike some grid systems, Grid does not define a container size. Instead, rows will expand to fill the available space. This allows you to use Grid for an entire page, a page container, or even a single section of a page, as needed.

There are three different grid types you can use: 12-column, 16- column or 24-column. You can also use overrides (see below) if you need different grid types in different places. This page uses a 12-column grid, defined in grid-12.css.

Rows are defined using the .row class. Use rows to horizontally align a set of components. Inside each row, mark blocks as .cell-1 through .cell-12 to configure how many columns that cell will consume.

Push/Pull

You can use .push- and .pull- classes to reorder the visual display of cells in a row. For example, .push-2 applied to a cell will move it two columns to the right. Push and pull will often be used in pairs, to swap cell positions, but can also be used to create row-level indentation.

Centered

The .center class can be applied to a cell of any size to center the cell in the row. It works best if the centered cell is the only content of the row; if there are any other cells, they should be defined before the centered cell, and pushed/pulled into the correct position.

Nesting

Grids can be nested to arbitrary depth; just put a .row inside another .row. To remove the outer margin, add the .inner class to the row.

Responsive

There are screen-size-prefixed versions of all of the normal .cell-, .push- and .pull- classes. These can be used to override the size or position of a cell when at a specific screen size. For example, a cell could be marked as .cell-4 .cell-s-12 to take up the full width on small screens, such as mobile phones, or a third of the width on larger screens.

Each screen size also has .show- and .hide- classes to display or hide content, respectively,

Overrides

Only one grid-X.css stylesheet can be used on a page. If more than one grid style is needed, you can link additional grid-X-override.css stylesheets. These provide .row-X and .grid-X classes that can be used to override the number of columns for a specific row or entire section, respectively.

Support

Grid makes use of the box-sizing CSS attribute in order to keep the size down; unfortunately, this means only IE 8+ is supported.
